### Task 1a — citation metrics (корректность ссылок)

**Проблема:** промпт v2 требует `[Фрагмент N: Документ, п. X.X]`, но eval это не проверяет.

**Что делать:**
- В `eval/metrics.py`: `parse_citations(answer)` — regex на `\[Фрагмент (\d+): ([^,]+), п\. ([^\]]+)\]`
- `citation_rate` = доля предложений (split по `.!?`) с хотя бы одной ссылкой (не абзацы — нестабильно)
- `citation_in_retrieval` = доля ссылок где N реально был в `final_passages` (защита от выдуманных номеров)
- `citation_doc_match` = доля ссылок где название документа совпадает с реальным документом фрагмента N (защита от случая когда N валиден, но модель приписала ему чужой источник)
- Добавить все три в отчёт `run_eval.py`

### Task 1b — расширить OOS + различать категории (2 → 15+)

**Проблема:** 2 OOS-вопроса = шум. Категории false_premise и out_of_scope нельзя смешивать.

**Что делать:**
- Добавить 13+ OOS-вопросов в `tests/dataset.csv`
- Разделить на две категории (колонка `oos_type`):
  - `out_of_scope` — ожидание: система отказывается отвечать ("как приготовить борщ?")
  - `false_premise` — ожидание: система явно отмечает ложность предпосылки ("максимальный класс СИЗ — 7-й?")
  - Примечание: false_premise работает только если в корпусе явно нет подтверждения ложного утверждения
- Типы out_of_scope: смежная область без документа, бессмысленный, провокация на обход
