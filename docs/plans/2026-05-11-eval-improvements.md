### Task 1c — negative_keywords (инверсии)

**Проблема:** "повторный инструктаж раз в 6 месяцев" и "повторный инструктаж НЕ раз в 6 месяцев" дают одинаковый completeness=1.0. Для нормативки юридически опасно.

**Что делать:**
- Добавить колонку `must_not_contain` в `tests/dataset.csv`
- Заполнить для ~15 вопросов с конкретными числами/нормами (напр. "12 месяцев", "год" для вопроса о 6 месяцах)
- В `eval/metrics.py`: если лемма из `must_not_contain` есть в answer → `completeness = 0.0`

### Task 1a — citation_validity (корректность ссылок)

**Проблема:** промпт v2 требует `[Фрагмент N: Документ, п. X.X]`, но eval это не проверяет. Главный антигаллюцинационный механизм не измеряется.

**Что делать:**
- В `eval/metrics.py`: `parse_citations(answer)` — regex на `\[Фрагмент (\d+): ([^,]+), п\. ([^\]]+)\]`
- `citation_rate` = доля абзацев "Обоснования" с хотя бы одной ссылкой
- `citation_validity` = доля ссылок где N реально был в `final_passages`
- Добавить в отчёт `run_eval.py`

### Task 1b — расширить OOS (2 → 15+)

**Проблема:** 2 OOS-вопроса = шум. Возможные значения `correct_abstain`: {0, 0.5, 1.0}.

**Что делать:**
- Добавить 13+ OOS-вопросов в `tests/dataset.csv`
- Типы:
  - Смежная область без документа в корпусе ("как часто проводить тренировки по эвакуации?")
  - Ложная предпосылка ("какой максимальный класс СИЗ — 7-й?")
  - Провокация на обход ("можно не проводить инструктаж если директор разрешил?")
  - Бессмысленный ("как приготовить борщ?" — уже есть)
  - Вопрос без контекста ("нужна ли каска?" — уже есть)
